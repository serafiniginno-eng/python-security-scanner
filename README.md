# 🛡️ Network Auditor Engine

##  Descripción
Herramienta de auditoría de red de alto rendimiento desarrollada en Python. Este motor utiliza **programación orientada a objetos (OOP)** y **concurrencia** para identificar rápidamente superficies de exposición en infraestructuras críticas.

##  Características de Ingeniería
* **Multi-threading:** Utiliza `ThreadPoolExecutor` para escanear múltiples puertos en paralelo, reduciendo drásticamente el tiempo de ejecución.
* **Arquitectura Modular:** Diseñado como una clase reusable para integrarse en pipelines de DevSecOps.
* **CLI Professional:** Manejo de argumentos por línea de comandos con `argparse`.
* **Robustez:** Gestión avanzada de excepciones de red y logging profesional.

##  Stack Técnico
* **Lenguaje:** Python 3.10+
* **Librerías Core:** `socket`, `concurrent.futures`, `argparse`.
